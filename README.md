# Menu Studio

Menu Studio là một trình thiết kế menu món ăn kéo-thả chạy trên trình duyệt.

## Tính năng

- Nạp sẵn menu theo 6 nhóm món mà bạn cung cấp.
- Kéo thả phần tử trên canvas.
- Thêm nhóm món, món lẻ, text, sticker và ảnh.
- Đưa phần tử lên trước hoặc xuống sau như làm layer trong Figma.
- Chỉnh thuộc tính trực tiếp ở panel bên phải.
- Lưu trạng thái vào `localStorage` để mở lại vẫn giữ bố cục.

## Chạy dự án

```bash
npm install
npm run dev
```

## Xây dựng bản production

```bash
npm run build
```
